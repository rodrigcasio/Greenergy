## Hackathon Project & Problem Statement

This project was developed as part of a university hackathon. The primary challenge addressed is the excessive production of carbon dioxide (CO₂) and its negative impact on our environment. 

**Greenergy** aims to help companies and their employees become more conscious of their daily habits, reduce their carbon footprint, and contribute to a healthier ecosystem by promoting sustainable practices and cutting down pollution.

---

## What is Greenergy?

Greenergy is a modern web application designed to help businesses and their employees understand and improve their environmental impact. The platform offers a straightforward and interactive way for employees to evaluate their daily habits and see how they contribute to their company's overall sustainability goals.

---

## Key Features

- **User Management:** Secure employee registration and account creation.
- **Environmental Assessment:** A dynamic 10-question quiz tracking daily consumption and habits.
- **Instant Feedback:** Tailored recommendations and sustainability insights generated instantly based on quiz answers.
- **Progress Tracking:** Interactive historical logs allowing users to monitor their score evolution over time.
- **Corporate Analytics Dashboard:** High-level impact summaries and analytics for authorized company administrators.
- **Responsive Design:** A clean, mobile-friendly interface tailored to match corporate branding.

---

## 🚀 How to Use (Original Workflow)

1. Register with your name, email address, and a secure password.
2. Complete the interactive environmental assessment.
3. Review your personalized sustainability score and action items.
4. Log back in at any time to monitor your progress or retake the assessment.

## 💻 Local Installation & Setup

If you want to clone this repository and run the application locally on your machine (using an environment like XAMPP, MAMP, or Laragon), follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```
2. **Move web files:**
   Place the project files into your local server's root directory (e.g., the `htdocs` folder in XAMPP).
3. **Import the Database:**
   - Open **phpMyAdmin** on your local machine.
   - Create a brand new database named `greenerg_db`.
   - Click the **Import** tab, choose the `.sql` backup file included in this repository, and run it.
4. **Configure Environment Connection:**
   Open the database configuration file (e.g., `config.php` or `.env`) and update the credentials to match your local database settings (typically user: `root` with no password).
